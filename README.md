# https://ez.chips.gg

## Dependancies
* [Parcel](https://parceljs.org)
* [Chips SDK](https://chips.gg)
* [Node.JS](https://nodejs.org)

## How do I run this?

### Create a .env

```
SOCKET_URL=wss://ezsocket.chips.gg
AUTH_URL=https://ezauth.chips.gg
```

### Start the app

```
yarn && yarn start
```
